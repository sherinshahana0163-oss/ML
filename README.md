# Loan Eligibility Prediction System

Machine learning project to predict loan approval based on applicant financial and personal details.

## Tools & Skills

* Python
* Pandas
* NumPy
* Scikit-learn
* Machine Learning
* EDA
* Matplotlib

## Project Workflow

* Data Cleaning
* Data Preprocessing
* Exploratory Data Analysis
* Model Training
* Hyperparameter Tuning
* Model Evaluation

## Best Model

Gradient Boosting

## Key Insights

* Credit score strongly influences approval
* Income and loan amount affect eligibility
* Balanced data improved model accuracy

## Outcome

Built a predictive model for automated loan approval decision-making.
